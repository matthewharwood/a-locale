# Bootstrap
`yarn install`

# To run App
`npm start`

# To run tests:
- Run once
    - `npm run test:single`
- Run watch
    - `npm run test`

# To view coverage:
- In dev:
    -`npm run coverage`
- After push (codecov)
    - https://codecov.io/gh/matthewharwood/a-locale

# CI
- https://travis-ci.org/matthewharwood/a-locale

# To Contribute:
- please follow:
    - https://egghead.io/lessons/javascript-introduction-to-how-to-write-an-open-source-javascript-library
    - `git add -A`
    - `npm run commit`
    - `git push`
    - watch https://travis-ci.org/matthewharwood/a-locale
